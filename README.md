# AeroPINN Aircraft

Physics-Informed Neural Networks for Aircraft Wing Aerodynamic Analysis using FastAPI + Next.js + PyTorch

## Project Status

- ✅ Backend: FastAPI with Physics-Informed Neural Networks
- ✅ Frontend: Next.js with React Three Fiber 3D visualization
- ✅ GitHub Repository configured
- ✅ Vercel Frontend Deployment configured (Root directory: frontend)
- ⏳ Railway Backend Deployment (pending)
- ⏳ Comet.ml Integration (pending)

## Features

- Physics-Informed Neural Networks for wing aerodynamic analysis
- 3D wing geometry visualization with React Three Fiber
- Support for CSV file uploads with aerodynamic data
- Real-time aerodynamic coefficient calculations (CL, CD, Cm)
- Advanced physics equations: Continuity, Navier-Stokes, Kutta condition
- Input parameters: Angle of Attack, Reynolds number, Mach number
